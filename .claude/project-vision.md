# Universal Experience - プロジェクト構想

## 🎯 プロジェクトミッション

**「すべての感覚を、すべての人に。」**

Universal Experienceは、障害者体験の「決定版」アプリケーションです。色覚障害、聴覚障害など、複数の感覚障害をシミュレーションし、世界中の人々がアクセシビリティの重要性を理解し、共感を深めることを目的としています。

## 💡 コンセプト

### 3つの「ユニバーサル」

1. **世界共通（Universal Global）**
   - 全世界で使用可能なアプリケーション
   - 言語・文化を超えた体験の提供
   - グローバルスタンダードとしての地位確立

2. **統一的存在（Universal Standard）**
   - 乱立する障害体験シミュレータを統一
   - 科学的に正確なアルゴリズムの採用
   - 医学的検証に基づいた信頼性

3. **包括的対応（Universal Coverage）**
   - 複数の障害タイプに対応
   - 視覚、聴覚、運動機能など広範囲をカバー
   - 段階的な機能拡張

## 📱 プラットフォーム戦略

### サポート対象
- ✅ Android 6.0+ (API 23+)
- ✅ Windows 10/11
- ⬜ macOS 10.14 (Mojave)+
- ⬜ Linux (Ubuntu 20.04+, Fedora 34+)

### 非サポート
- ❌ iOS
  - **理由**: Appleのサンドボックス制約により、システム全体への色覚フィルタ適用が技術的に困難
  - **代替**: iPadOS/iOSの標準アクセシビリティ機能を案内

### Web版の方針
- **不採用**
  - システム全体のフィルタが必要
  - ブラウザ内のみのシミュレーションは目的に不適合
  - デスクトップアプリとして完結させる

## 🌟 差別化要因

### 既存製品との違い

| 特徴 | Universal Experience | 既存シミュレータ |
|------|---------------------|---------------|
| 対応障害 | 複数タイプ（拡張予定） | 単一タイプが多い |
| 適用範囲 | OS全体 | アプリ内/ブラウザ内 |
| プラットフォーム | マルチプラットフォーム | 限定的 |
| アルゴリズム | LMS変換（科学的） | 簡易的な処理 |
| オープンソース | Yes（予定） | クローズドが多い |
| 常駐機能 | Yes | 限定的 |
